## Doctor-hygiene scan

Чтобы защитить doctor-контекст от регрессий по безопасности данных, сетевым вызовам и недетерминированному времени, в репозитории работает автоматический сканер запрещённых паттернов.

### Что сканируется

- `src/pages/doctor/**` — страницы и логика рабочего места врача.
- `src/App.tsx` — корневой роутинг.

Тестовые файлы (`*.test.ts(x)`, `*.hygiene.test.*`) исключаются автоматически.

### Запрещённые паттерны

Единый источник правды — [`scripts/forbidden-patterns.mjs`](./scripts/forbidden-patterns.mjs). Этот же список используют git-хуки, GitHub Actions и vitest-тест `VisitImagingTab.hygiene.test.ts`.
